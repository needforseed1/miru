Bundled uosc 5.12.0
===================

This directory contains the uosc mpv on-screen controller from:

https://github.com/tomasklaen/uosc

The optional ziggy helper binaries are intentionally not bundled. The app uses
uosc for playback controls only and keeps thumbnail/extra helper behavior out of
the default playback path.
